Fallback Video Test

To test the fallback functionality:

  1. Open the browser developer console (F12)
  2. Go to Network tab
  3. Right-click on the analytics API request and block the domain
  4. Or temporarily modify the API URL in trending-video-player.js
  5. Refresh the trending page

Expected Behavior:

  • ✓ No error messages displayed to users
  • ✓ 15 fallback videos play smoothly
  • ✓ "📡 Showing popular videos" indicator appears
  • ✓ "Try Nostr Direct →" link is available
  • ✓ Console shows retry attempts every 30 seconds
  • ✓ All player controls work (play/pause/next/prev)
  • ✓ Videos loop 4 times before auto-advancing
  • ✓ Swipe gestures work on mobile
